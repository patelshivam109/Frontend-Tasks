# CodeAlpha Internship Tasks

This repository contains the projects completed as part of the CodeAlpha internship tasks.  
The projects are developed using web technologies such as HTML, CSS, and JavaScript.

---

## 📁 Projects Included

### 🔹 Task 3: Build Your Portfolio Site
A personal portfolio website designed to showcase skills, projects, resume, and contact information.

#### Features
- Clean and responsive design
- Sections for About, Skills, Projects, and Contact
- Smooth scrolling and animations
- Hover effects for better UI experience
- Deployment-ready (GitHub Pages / Netlify)

#### Technologies Used
- HTML5
- CSS3
- JavaScript



---

### 🔹 Task 4: Music Player using JavaScript
A web-based music player application with interactive audio controls.

#### Features
- Play, Pause, Next, Previous controls
- Displays song title, artist, and duration
- Progress bar to track song playback
- Volume control
- Playlist and autoplay functionality

#### Technologies Used
- HTML5
- CSS3
- JavaScript

3. Open `index.html` in any modern web browser
